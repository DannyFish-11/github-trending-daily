## Detailed Description

OpenClaw是一个个人AI助手，可以在用户自己的设备上运行。它支持多种消息渠道，包括WhatsApp、Telegram、Slack、Discord、Google Chat、Signal、iMessage、Microsoft Teams、WebChat，以及BlueBubbles、Matrix、Zalo和Zalo Personal等扩展渠道。它可以在macOS/iOS/Android上进行语音交流，并能渲染一个可控的实时Canvas。Gateway是控制平面，而产品本身是助手。

如果你想要一个感觉本地化、快速且始终在线的个人单用户助手，OpenClaw就是你的选择。

核心特性包括：
*   **本地优先的Gateway** — 会话、渠道、工具和事件的单一控制平面。
*   **多渠道收件箱** — 支持WhatsApp、Telegram、Slack、Discord、Google Chat、Signal、BlueBubbles (iMessage)、iMessage (legacy)、Microsoft Teams、Matrix、Zalo、Zalo Personal、WebChat、macOS、iOS/Android。
*   **多代理路由** — 将入站渠道/账户/对等方路由到隔离的代理（工作区+每个代理的会话）。
*   **语音唤醒 + 对话模式** — 适用于macOS/iOS/Android的始终在线语音功能，支持ElevenLabs。
*   **实时Canvas** — 由代理驱动的视觉工作区，支持A2UI。
*   **一流的工具** — 浏览器、Canvas、节点、Cron、会话以及Discord/Slack操作。
*   **配套应用程序** — macOS菜单栏应用程序 + iOS/Android节点。
*   **入门引导 + 技能** — 带有捆绑/管理/工作区技能的向导式设置。

## Tech Stack

## Use Cases

1. 个人AI助手：在各种消息平台上提供智能回复和自动化服务。
2. 多渠道通信管理：统一管理来自WhatsApp、Telegram、Slack等多个平台的对话。
3. 设备本地化操作：在macOS/iOS/Android设备上执行语音唤醒、屏幕录制、相机控制等本地任务。
4. 开发者工具：通过CLI、Gateway、Agent等工具进行开发和集成。
5. 自动化工作流：利用Cron、Webhooks和各种技能实现任务自动化。
